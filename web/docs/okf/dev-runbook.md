# Dev & build runbook

Vite + React 19 SPA. Node + npm. Entry: `index.html` → `src/main.tsx` → [`<App/>`](/app-shell.md).

# Examples

```bash
npm install
npm run dev       # vite dev server, --host 0.0.0.0
npm run build     # tsc -b && vite build → dist/
npm run preview   # serve the production build, --host 0.0.0.0
```

# Citations

- Scripts: `package.json:6`.
- TypeScript project refs: `tsconfig.json` → `tsconfig.app.json` / `tsconfig.node.json`.
- Vite config: `vite.config.ts` (React plugin).
- Runtime terrain/route/texture assets are expected under `public/assets/<valley-id>/` (served at `/assets/...`); see [mock data](/mock-data.md) for the file list. They are generated by an upstream pipeline, not built here.
